    **1. Create `src/server/agent/setup.ts` — Agent factory:**

    Export a `createAgent` function that takes `{ provider, modelId, apiKey, history }` and returns an `Agent` instance.
    - Import `Agent` from `@mariozechner/pi-agent-core`
    - Import `getModel` from `@mariozechner/pi-ai`
    - The function creates an Agent with:
      - `initialState.systemPrompt`: `"You are a helpful assistant."`
      - `initialState.model`: result of `getModel(provider, modelId as any)`
      - `initialState.tools`: `[]` (no tools in Phase 2)
    - `getApiKey`: callback returning the provided `apiKey` string
    - Return the Agent instance
    - Type the parameters: `provider: "anthropic" | "openai"`, `modelId: string`, `apiKey: string`

    **2. Create `src/server/lib/stream-adapter.ts` — AgentEvent to SSE mapper:**

    Export an `adaptAgentEvents` function that takes `{ agent, stream, onDone }` where:
    - `agent` is an `Agent` instance
    - `stream` is the Hono SSE stream (type from `hono/streaming`)
    - `onDone` is a `() => void` callback to signal completion

    The function:
    - Calls `agent.subscribe((event) => { ... })` to listen for events
    - On `event.type === "message_update"`: check `event.assistantMessageEvent.type === "text_delta"`, then call `stream.writeSSE({ event: "text_delta", data: JSON.stringify({ data: event.assistantMessageEvent.delta }) })`
    - On `event.type === "agent_end"`: call `stream.writeSSE({ event: "done", data: "{}" })`, then `unsubscribe()`, then `onDone()`
    - Wrap all `stream.writeSSE` calls in try/catch (stream may be closed)
    - Return the `unsubscribe` function

    Import `AgentEvent` type from `@mariozechner/pi-agent-core`.

    **3. Create `src/server/routes/chat.ts` — POST /api/chat:**

    Follow the existing Hono route pattern from `auth.ts`:
    - Import `Hono`, `streamSSE` from `hono/streaming`, `createAgent` from `../agent/setup`, `adaptAgentEvents` from `../lib/stream-adapter`, `getCredentials` from `../lib/credentials`
    - Create `chatRoutes = new Hono()`
    - Add `chatRoutes.post("/", async (c) => { ... })`:
      1. Parse JSON body: `{ message, model, provider, history }` where `history` is `Array<{ role: string; content: string }>`
      2. Get `apiKey` via `getCredentials(provider)` — if null, return `c.json({ error: "Not authenticated" }, 401)`
      3. Return `streamSSE(c, async (stream) => { ... })`:
         - Call `createAgent({ provider, modelId: model, apiKey })`
         - Create a `done` Promise with the Promise anchor pattern:
           ```
           const done = new Promise<void>((resolve) => {
             const unsubscribe = adaptAgentEvents({ agent, stream, onDone: resolve });
             stream.onAbort(() => { agent.abort(); unsubscribe(); resolve(); });
           });
           ```
         - Build the prompt message. If `history` array is provided and non-empty, call `agent.prompt(message, { history })`. If no history, call `agent.prompt(message)`. NOTE: Check the Agent.prompt() signature — it may accept just a string, or string + options. If history is not a direct parameter, the message itself may need to include context. Start with just `agent.prompt(message)` and verify behavior.
         - Catch errors from `agent.prompt()`: write an SSE error event `{ event: "error", data: JSON.stringify({ message: err.message || "Agent error" }) }`, wrapped in try/catch
         - `await done` to anchor the stream
    - Export `{ chatRoutes }`

    **4. Update `src/server/index.ts` — Register chat route:**

    Add `app.route("/api/chat", chatRoutes)` after the existing `app.route("/api/auth", authRoutes)` line.
