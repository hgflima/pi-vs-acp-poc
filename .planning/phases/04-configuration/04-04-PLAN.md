Drag & drop handling:

- `onDrop`: preventDefault, set dragging false. Extract path from dropped items. For web apps, dropped folders come as `DataTransferItem` with `webkitGetAsEntry()`. - For simplicity in this POC: on drop, extract the folder name from the drop event and populate the input field. The user confirms or edits the path manually. This avoids the browser security limitation of not exposing full filesystem paths.

- "Change Folder" click: call onDirectoryChange("") to reset to empty state.

Input handling:
- Input below drop zone with "Enter" key submission.
- On Enter or button click: call `onDirectoryChange(inputValue)`.
