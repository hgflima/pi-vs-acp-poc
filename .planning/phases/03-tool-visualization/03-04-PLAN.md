    1. Start the dev server: `npm run dev`
    2. Open http://localhost:5173 in the browser

    Test 1 -- Bash tool (terminal card):
    Send: "Run `echo hello world` in bash"
    Expected: A dark terminal-style card appears inline with `$ echo hello world` as the command and `hello world` as the output. The card shows a spinning icon while running, then a green checkmark when done.

    Test 2 -- File read tool (file card):
    Send: "Read the file package.json"
    Expected: A green-tinted card appears with the file path `package.json` in the header and the file contents displayed below with monospace font.

    Test 3 -- List files tool (search card):
    Send: "List the files in the src directory"
    Expected: A blue-tinted card appears with "src" in the header and a list of files/directories below.

    Test 4 -- Interleaving:
    Send: "First read package.json, then tell me what the project name is"
    Expected: The assistant's response shows text, then a file card, then more text explaining the project name. The text and tool card are interleaved, not separate messages.

    Test 5 -- Running state visibility:
    Send: "Run `sleep 2 && echo done`"
    Expected: The terminal card appears immediately with a spinning blue icon. After ~2 seconds, the icon changes to a green checkmark and "done" appears in the output area.

    Test 6 -- Error handling:
    Send: "Read the file /nonexistent/path/file.txt"
    Expected: A file card appears with an error state -- red X icon and error message about file not found.
