2. What type of app is it? (mobile, web, SaaS, etc.)

**About the Users:**
4. Who are the target users? Are there multiple personas?
5. What problem does the app solve for them?

**About the Journey:**
6. Which journey should be mapped? (onboarding, purchase, daily use, etc.)
7. Is this a current-state or future-state ("to-be") journey?
8. What stages does the user go through?

**About the Output:**
9. What level of detail do you need?
10. Should the map include emotional states, touchpoints, channels, or backend processes?
